The *keyword* `let` allows us creating block scopes in JavaScript. Let's see some examples.

## Block scope

Using `var`.

```js
var me = 'James Bond';

if(true) {
  var me = 'Chuck Norris';
}

console.log(me);
// Chuck Norris
```

Using `let`.

```js
let me = 'James Bond';

if(true) {
  let me = 'Chuck Norris';
}

console.log(me);
// James Bond
```

## Loops

Using `var`.

```js
for(var i = 0; i < 5; i++) {
  // ...
}

console.log(i);
// 5
```

Using `let`.

```js
for(let i = 0; i < 5; i++) {
  // ...
}

console.log(i);
// ReferenceError: i is not defined
```

## Duplidated variables

Using `var`.

```js
var country = 'Brazil';
//..
//..
//..
var country = 'Belgium';

console.log(country);
// Belgium
```

Using `let`.

```js
let country = 'Brazil';
//..
//..
//..
let country = 'Belgium';

console.log(country);
// Duplicate declaration 'country'
```
